# 餐廳外場模擬器

一個類似 Overcooked 的餐廳外場模擬器，重點在於除了滿足客人的食物需求，還要在情緒上讓每個人滿意。

## 操作方式

### 移動
- **W/↑**: 向上移動
- **S/↓**: 向下移動
- **A/←**: 向左移動
- **D/→**: 向右移動

### 表情控制
- **J**: 開心表情 😊
- **K**: 中性表情 😐
- **L**: 難過表情 😢

**注意**: 使用表情時會持續消耗 SAN 值！

## 遊戲機制

### 客人系統
- **3 種個性**:
  - 友善 (Friendly): 喜歡開心表情
  - 中性 (Neutral): 喜歡中性表情
  - 暴躁 (Grumpy): 喜歡難過表情（需要同情）

### 食物系統（台灣在地風格）
1. 牛肉麵
2. 臭豆腐
3. 珍珠奶茶
4. 蚵仔煎
5. 滷肉飯

### 內場人員
- **內場人員 1**: 供應牛肉麵、臭豆腐、珍珠奶茶（喜歡開心表情）
- **內場人員 2**: 供應蚵仔煎、滷肉飯（喜歡中性表情）

### QTE 點單系統
1. 靠近客人並按下任意表情鍵開始點單
2. 對話泡泡會顯示隨機的 emoji 或食物名稱，快速輪流顯示
3. 當顯示到客人想要的**食物名稱**時，鬆開表情鍵即可接單成功
4. 如果做錯表情，客人滿意度會下降
5. 如果 QTE 失敗，需要重新開始

### SAN 值系統
- 使用表情時會持續消耗 SAN 值
- SAN 值過低會影響遊戲體驗

### 排隊系統
- 客人會陸續加入排隊
- 最多同時 5 位客人
- 每 5 秒生成一位新客人
